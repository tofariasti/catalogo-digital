## 🚀 Como Usar Este Projeto

### Para Visualizar
1. Acesse: https://tofariasti.github.io/catalogo-digital/
2. Teste os filtros de categoria
3. Clique nos botões WhatsApp
4. Teste em diferentes dispositivos

### Para Clonar e Personalizar
```bash
# Clone o repositório
git clone https://github.com/tofariasti/catalogo-digital.git
cd catalogo-digital

# Abra no navegador
open index.html

# Ou use um servidor local
python -m http.server 8000
# Acesse: http://localhost:8000
```

### Para Personalizar
1. **Produtos**: Edite `site/index.html` → seção `.products-grid`
2. **Cores**: Altere `:root` no `<style>`
3. **WhatsApp**: Substitua o número `5551989030405`
4. **Textos**: Modifique títulos e descrições
